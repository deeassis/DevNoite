Plano de Estudos Semanal

Objetivos da Semana

  • Revisar conceitos de HTML e CSS
  • Praticar JavaScript básico
  • Desenvolver um projeto simples em HTML e CSS

Tarefas Diárias

  1. Segunda-Feira: Ler sobre HTML e praticar exercícios.
  2. Terca-feira: Estudar CSS e criar uma página de estilo.
  3. Quarta-Feira: Praticar JavaScript e resolçver desafios.
  4. Quinta-Feira: Continuar o projeto e revisar conceitos.
  5. Sexta-Feira: Trabalhar no design do projeto.
  6. Sábado: Revisar todo o material estudado durante a semana.
  7. Domingo: Refazer exercícios e preparar se para a próxima semana.

Material Necessário

  • Conexão com Internet
  • Computador(Desktop ou Notebook)
  • Visual Studio Code instalado
  • Mecanismos de pesquisa Web (Google, ChatGPT entre outros)
  • Caderno para anotações

Dúvidas e Definições

O que é HTML?
HTML é uma linguagem de marcação de hipertexto utilizada na criação de documentos e páginas da web. Os marcadores, chamados de “tags”, servem para indicar a função de cada elemento na página, como textos, imagens e vídeos, além de suas conexões com outros elementos (links) e como eles serão interpretados pelo navegador.
Qual a diferença entre HTML e CSS?
O Html faz a ligação entre diversos elementos, criando uma estrutura que sustenta a página. Enquanto isso o CSS permite alterar a aparência do site em mais detalhes e aplicar uma série de efeitos e estilos que deixam os sites mais dinâmicos e agradáveis.
Como posso aprender JavaScript?
Para aprender JavaScript, comece entendendo os conceitos básicos de programação e se familiarize com HTML e CSS, já que o JavaScript é frequentemente usado para manipular elementos de uma página web. Em seguida, aprenda os fundamentos do JavaScript, como variáveis, loops, funções e o DOM. Pratique criando pequenos projetos, como uma calculadora ou um jogo simples, e explore recursos de aprendizado, como cursos online e a documentação do MDN Web Docs. À medida que avança, participe de comunidades de desenvolvedores, trabalhe em projetos pessoais, e eventualmente explore frameworks como React ou Node.js para ampliar suas habilidades.
